# PhotoBooks Gallery

Онлайн редактор для создания фотокниг с личным кабинетом пользователя.

## Поддерживаемые форматы фотокниг

- 15×20 см (стандартный)
- 20×20 см (квадратный)
- 20×25 см (средний альбомный)
- 20×30 см (альбомный)
- 25×25 см (большой квадратный)
- 30×30 см (премиум квадратный)
- 30×40 см (премиум альбомный)

## Характеристики

- Количество разворотов: 10-100
- Тип бумаги: матовая
- Тип обложки: твёрдая с возможностью печати
- Дополнительные опции: подготовка для тиснения (в разработке)

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте переменные окружения в файле .env

3. Создайте базу данных:
```bash
alembic upgrade head
```

4. Запустите сервер:
```bash
uvicorn app.main:app --reload
```
